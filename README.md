# Oracle Example

Example source code for O<sub>byte</sub> oracle.  This oracle periodically posts stock prices of Apple and Facebook.  Copy its source code and plug in another data source.

## Requirements

node.js 8+

## Install
```
npm install
```
## Run
```
node start.js
```
